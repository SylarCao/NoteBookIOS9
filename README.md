# NoteBookIOS9

#### 配置环境
* iphone6s + iOS9
* 其他适配可能有问题，可能有bug
<br/>

#### 版本目前功能
* 参照苹果自带的备忘录和提醒事项
* 有像svn或者git一样的版本控制功能（仅限本地）
* 提醒事项，本地推送，数字永远不会消失
- 应用图标的红点存在用户就肯定会去点
* 增加九宫格密码功能
